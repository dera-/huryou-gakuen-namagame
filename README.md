# 不良学園！校舎裏の戦い

校舎裏を舞台に、不良たちをなぎ倒してスコアを稼ぐ 2D アクションゲームです。  
プレイヤーは主人公を操作し、通常攻撃・強攻撃・アイテム強化を使いながら 100 秒間生き残って戦います。

## ゲームの流れ

- タイトル画面: 15 秒
- ゲーム本編: 100 秒
- 結果発表画面

本編の進行は以下の 3 段階です。

1. `チンピラ掃討`
2. `番長戦`
3. `超乱戦！とにかく倒せ！`

## ルール

- 画面ドラッグで主人公を移動します。
- 敵を射程内に入れて敵をタップすると通常攻撃します。
- 敵を 0.5 秒長押しすると強攻撃します。
- 強攻撃中は主人公の下にゲージが表示されます。
- 強攻撃は通常攻撃の 3 倍ダメージです。
- 強攻撃で敵を倒すと撃破スコアが 2 倍になります。
- アイテムは射程内に入るだけで取得できます。
- 焼きそばパンを取ると HP が 35 回復します。
- 木刀を取ると攻撃力が 20% 上がります。
- サングラスを取ると射程角度が 15 度広がります。
- HPが0になっても再挑戦可能です。

## 起動方法

初回のみ依存パッケージをインストールします。

```sh
npm install
```

次のどちらかでローカル実行できます。

```sh
npm start
```

または

```sh
npm run start:multi
```

## 使用アセット

- 画像: `image/`
- 音声: `audio/`
- メインスクリプト: `script/main.js`

アセットを追加・変更したあとに `akashic scan asset` を実行すると、`game.json` を更新できます。
